About Us

Video to Audio Converter is a free, open-source tool built to help you extract audio from videos – without uploading anything.

Why We Built This

Most online converters require file uploads, which raises privacy concerns. We wanted a solution that:

  • Processes files 100% in your browser
  • Never stores or transmits your data
  • Works on any device
  • Is completely free

Tech Stack

  • Frontend: HTML, CSS, JavaScript
  • Audio Encoding: lamejs (MP3), Web Audio API (WAV)
